# 訪客（Visitor / Guest）

## 定義

尚未登入、正在瀏覽網站的潛在學員。透過站長以 `[pc_courses]` 短代碼嵌入的課程列表瀏覽平台課程，尋找感興趣的課程。

## 合法性

外部使用者（External Actor）—— 從瀏覽器發起 HTTP 請求，是 Activity / Feature 的合法觸發者。

## 能力與權限

| 能力 | 說明 |
|------|------|
| 瀏覽課程列表 | 透過嵌有 `[pc_courses]` 的頁面檢視 publish 狀態的課程卡片 |
| 翻頁瀏覽 | 點擊課程列表底部的傳統頁碼導航（‹ 1 2 3 4 ›），以 AJAX 載入其他頁課程 |
| 進入課程銷售頁 | 點擊課程卡片進入單一課程的銷售頁 |

無需登入、無需任何 WordPress capability。分頁 AJAX 端點對未登入訪客開放（`permission_callback => null`），僅回傳 publish 且 visible 的課程，不洩漏非公開資料。

## 相關流程
