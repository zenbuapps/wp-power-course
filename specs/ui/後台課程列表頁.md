# 後台課程列表頁（Power Course Admin SPA）

管理員在 WordPress 後台 Power Course 的課程列表，管理所有課程。React SPA（Refine.dev + Ant Design），
資料來源為 REST `GET /courses`。

## 描述

- 表格逐列呈現課程，欄位含名稱、類型、狀態、銷售數、價格、庫存、開課時間、分類標籤、操作。
  - 元件：`js/src/pages/admin/Courses/List/`（`Table/`、`hooks/useColumns.tsx`）。
- 後台預設回傳並顯示 `publish` / `draft` / `future`（排程中）/ `private`（私密）四種非垃圾桶狀態課程（Issue #256）。

## 關鍵狀態

| 課程狀態 | 標籤呈現 |
|----------|----------|
| publish（已發佈） | 綠色「已發佈」Tag |
| draft（草稿） | 灰色「草稿」Tag |
| future（排程中） | **藍色「排程中」Tag**，沿用課程公告 `CourseAnnouncement/StatusTag.tsx` 的 scheduled 樣式與用語（Q4）；標籤旁直接顯示預計上架日期時間文字，例如「排程中 · 2025-08-01 09:00 上架」（Q3） |
| private（私密） | 灰色「私密」Tag |

## 篩選

- 狀態篩選器（`js/src/utils/constants.ts` `statusOptions`）新增「排程中(future)」選項（Q2）；
  管理員可單獨檢視排程中的課程，盤點即將上架的內容（旅程 3）。
- 既有選項：已發佈 / 待審 / 草稿 / 私密。

## 互動

- 點課程名稱 / 編輯 → 進入課程編輯頁；排程中的課程可直接編輯內容、調整發佈時間或取消排程，全程留在後台（旅程 2）。
