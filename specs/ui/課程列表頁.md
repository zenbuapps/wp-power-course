# 課程列表頁（[pc_courses] 短代碼）

站長以 `[pc_courses]` 短代碼嵌入任意頁面 / 文章的課程卡片列表。訪客可在此瀏覽 publish 課程並翻頁。

## 描述

- 課程卡片以 grid 排列（`inc/templates/components/list/pricing.php`，`card/pricing` 卡片），欄數由 `columns` 參數控制（預設 3）。
- 每頁顯示 `limit` 門課程（預設 12；`limit` 語意自 Issue #236 起改為「每頁數量」，可翻頁看完全部）。
- 列表底部為傳統頁碼導航區（`‹ 1 2 3 4 ›`），沿用前台既有 `.pc-pagination` 樣式（與留言分頁同款，見 `inc/assets/src/events/comment/components/Pagination.ts`）。

## 關鍵狀態

| 狀態 | 呈現 |
|------|------|
| 總頁數 > 1 | 顯示頁碼導航，目前頁套用 `current`、首頁時 prev 套 `disabled`、末頁時 next 套 `disabled` |
| 總頁數 = 1 | 不顯示頁碼導航 |
| AJAX 翻頁載入中 | 顯示 loading 狀態（卡片區或頁碼區），完成後抽換卡片內容 |
| 該頁無課程（總數為 0） | 顯示友善提示「目前沒有符合條件的課程」 |

## 互動

- 點頁碼 / prev / next → 純前端 AJAX 重新載入該頁課程卡片，不整頁刷新、不改變網址列 URL。
- 同一頁面若有多個 `[pc_courses]`，各自維護獨立的目前頁狀態，互不干擾。
