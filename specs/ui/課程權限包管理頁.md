# 課程權限包管理頁

## 描述
後台獨立管理頁（React 18 + Refine + Ant Design SPA），路徑 `js/src/pages/admin/AccessPasses/`。
管理員在此檢視、新增、編輯、停用、刪除「課程權限包（Access Pass）」。
權限包是可命名、可重用、可被多個商品共享的「權限資產」，定義一組課程觀看範圍。

## 關鍵屬性
- 列表：顯示所有權限包（名稱、範圍類型、期限模式、狀態、已掛載商品數）
- 新增／編輯表單：
  - 名稱（必填）
  - 範圍類型：全站 / 分類標籤 / 特定課程（三選一）
    - 分類標籤：多選 `product_cat` / `product_tag`（聯集，含子分類），表單需明確標示「此範圍為動態，含日後新增課程」
    - 特定課程：手動勾選固定課程清單（不隨新增課程擴張）
  - 期限模式：永久 / 跟隨訂閱 / 限時 N 天（limit_value + limit_unit）
- 停用：權限包不可再掛載到新商品，但已購用戶權限保留
- 刪除：二次確認對話框，並顯示「將影響 N 位已購用戶」的影響提示
- 範圍變更（縮小／改範圍）：儲存前 UI 警告「將影響 N 位已購用戶」
- 認證：WordPress 登入 Cookie + `X-WP-Nonce` Header
