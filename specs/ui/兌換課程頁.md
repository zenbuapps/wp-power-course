# 兌換課程頁

## 描述
供已登入用戶輸入票券代碼兌換課程的公開前台頁面（Issue #24）。
以 shortcode `[pc_redeem_ticket]` 嵌入任意頁面，支援 `?code=XXX` query string 預填。

## 關鍵元素
- 代碼輸入框（不區分大小寫；若帶 `?code=` 則自動預填）
- 兌換按鈕
- 結果訊息區：
  - 成功：「兌換成功」+ 前往教室入口頁 CTA
  - 兌換碼無效 / 已被使用 / 已失效 / 已擁有此課程的存取權 / 嘗試次數過多
- 未登入時：導向登入 / 註冊，登入後帶回本頁並保留已輸入代碼

## 資料來源
- `POST /power-course/tickets/redeem`

## 權限
- 需登入；未登入導向登入流程並保留代碼
