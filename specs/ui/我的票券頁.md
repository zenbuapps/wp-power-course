# 我的票券頁

## 描述
購買者在 WooCommerce「我的帳號」中查看自己所有課程分票票券的分頁（Issue #24）。
Endpoint：`/my-account/course-tickets/`（`add_rewrite_endpoint` + `woocommerce_account_menu_items`）。

## 關鍵元素
- 票券列表：課程名稱（來自 grant_payload.product_name / course_ids）、代碼、狀態（未兌換 / 已兌換 / 已作廢）
- 兌換者欄位：已兌換的票顯示兌換者、兌換時間
- 複製按鈕：複製票券代碼
- 分享連結按鈕：複製 `?code=XXX` 兌換連結（URL 分票）
- 狀態徽章：pending / redeemed / voided 對應顏色

## 資料來源
- `GET /power-course/tickets`（一般用戶僅回傳自己 purchaser_id 的票券）

## 權限
- 僅登入用戶可見，且只看得到自己購買的票券
